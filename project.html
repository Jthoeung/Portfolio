BeanThere

BeanThere is a full-stack web application that enables users to discover, browse, and rate local coffee shops. Built using React, JavaScript, HTML, and Node.js, the application features an interactive map that dynamically displays coffee shop locations through integrated web APIs. The project emphasized responsive design, user-friendly navigation, and seamless state management to deliver an intuitive user experience. Developed collaboratively using Git, BeanThere strengthened skills in front-end development, API integration, debugging, and cross-functional teamwork.
